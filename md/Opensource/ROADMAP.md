# Skillshelf Roadmap

This roadmap outlines the planned evolution of Skillshelf from a private library to a community-powered open-source platform.

## 🟧 Phase 1: Open-Source Foundation (Current)
- [ ] **Hybrid Content Architecture**: Migrate "Free" skills to Git-based storage while keeping "Premium" content in Supabase.
- [ ] **Community Documents**: Finalize `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, and Issue Templates.
- [ ] **Static Preview System**: Implement the image-first grid for ultra-fast browsing.
- [ ] **Tier System**: Add `tier` (Free, Gated, Premium) to the database schema.

## 🟨 Phase 2: User Experience & Engagement
- [ ] **Auth Gates**: Implement "Login to Preview" logic for gated skills.
- [ ] **Like & Save System**: Enable users to build personal collections.
- [ ] **Contributor Leaderboard**: Highlight community members who submit the best skills.
- [ ] **Advanced Search**: Filter by technical stack, design style, and complexity.

## 🟩 Phase 3: Scaling & Sustainability
- [ ] **Premium Subscription**: Integrate Stripe for access to exclusive, production-grade logic and designs.
- [ ] **Pro Dashboard**: Advanced tools for power users to manage their SKILL.md library.
- [ ] **API Access**: Allow developers to pull their saved skills directly into their own IDEs/workflows.
- [ ] **AI-Powered Validation**: Automatically test submitted skills for token accuracy and rendering consistency.
